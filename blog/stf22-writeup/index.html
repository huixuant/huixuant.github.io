STACK the Flags 2022: Hit You With That Challenge Writeup

hx | Dec 24, 2022

A while ago, I participated in GovTech’s STACK the Flags competition. One of the challenges I attempted was Hit You With That, but I didn’t manage to solve it within the duration of the CTF. I gave it a second try after the competition ended, and managed to find the flag with a bit of help. As part of the learning process, here is my attempt at documenting the steps I took to discover the flag.

Analysing PCAP in Wireshark

Opening up the given PCAP file in Wireshark, the first thing I noticed was a series of TCP packets. Following the stream, it appears that a HTTP GET request is made. The link opens up to the music video of BlackPink’s Ddu-Du Ddu-Du (which, I suppose, explains the challenge title). However, there isn’t any other useful information.

Next, sorting the packets by protocol, I noticed that there seemed to be a large number of ICMP packets that contained some sort of encoded data, which caught my interest.

Extracting Packet Data with Scapy

Using the scapy library, I wrote a simple Python script to exfiltrate the payload and dump it into a text file.

from scapy.all import *

pcap = rdpcap("STF22.pcapng") # reads pcap file 
listbytes = bytearray()
for i in range(len(pcap)):
    if pcap[i].haslayer(ICMP) and pcap[i][IP].dst == "192.168.96.11":
        listbytes.extend(pcap[i].load)

with open('bytes.txt', 'wb') as f:
    f.write(listbytes)

Decoding Payload

Viewing the extracted data in its entirety, it becomes clear that the payload was Base64 encoded. Hence, I used Cyberchef and loaded the From Base64 recipe to decode it. The output is a PNG file of BlackPink’s Pink Venom poster as seen below.

Finally, using the Strings recipe on Cyberchef gives us the flag we are looking for: STF22{Bl@ckPin9_V3n0m}.
